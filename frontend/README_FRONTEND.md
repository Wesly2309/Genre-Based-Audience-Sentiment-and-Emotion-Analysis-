Instruksi frontend (React + Vite + Tailwind)
============================================
1. Pastikan Node.js dan npm terpasang.
2. Masuk ke folder frontend:
   cd frontend
3. Install:
   npm install
4. Jalankan development:
   npm run dev
5. Build untuk produksi:
   npm run build
6. Setelah build, folder 'dist' akan dibuat. Salin folder dist ke backend/static (atau biarkan backend serve dari ../frontend/dist seperti konfigurasi).
